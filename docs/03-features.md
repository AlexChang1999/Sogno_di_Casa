# 03 — 完整功能說明

## 全站導覽列（所有頁面共用）

- **FORMA Logo** — 點擊回首頁
- **主選單**：首頁 / 商品 / 關於品牌 / 設計師
- **使用者名稱**（已登入時顯示）
- **商家後台**（僅管理員可見）
- **我的訂單**（連結至帳號頁）
- **登出** 按鈕
- **購物車圖示** + 即時商品數量計數（從 localStorage 讀取）

## 首頁 `index.html`

### Hero Banner 輪播
- 共 3 張全幅橫幅，自動輪播
- 每張含標題、副標、品牌名稱與 CTA 按鈕（「立即選購」/「發現更多」）

### 品牌跑馬燈
- 水平捲動展示品牌名稱：Herman Miller / Cassina / Vitra / Knoll / Fritz Hansen / Muji

### 精選商品區
- 兩個分頁 tab：**本季主打** / **全部商品**
- 每個 tab 顯示最多 4 件商品卡片
- 商品卡片含：圖片、品牌名稱、商品名稱、售價、「加入購物車」按鈕
- 由管理後台的「首頁展示」開關控制哪些商品出現在此

### Collection 區塊
- 兩個分類橫幅：**Living Collection（客廳）** / **Office Collection（辦公室）**
- 點擊「探索 →」連結至商品列表並帶入類別篩選

### 品牌精選區
- 標題「設計經典」，顯示 3 件商品
- 由管理後台「設計經典」開關控制

### 服務亮點
- 全台免費配送（滿 NT$50,000）
- 5 年品質保固
- 30 天鑑賞期
- 專屬設計顧問

### 頁腳
- 商品分類連結（客廳/臥室/辦公室/戶外）
- 品牌連結（Herman Miller / Cassina / Vitra / Knoll）
- 服務連結（配送說明/退換貨政策/常見問題/聯絡我們）
- 關於連結（品牌故事/永續設計/隱私政策）
- 版權聲明

## 商品詳情 `product-detail.html`

透過 URL hash 取得商品 ID，例如 `product-detail.html#5`

### 商品圖片區
- 主圖（hover 時放大顯示細節）
- 縮圖列（若有副圖則顯示，點擊切換主圖，少於 2 張時隱藏）
- 切換顏色選項時主圖連動替換

### 商品資訊區
- 品牌名稱
- 商品名稱
- 評價星級 + 評價總數（例：★★★★☆ 128 則評價）
- 售價（NT$ 格式）
- 設計師簡介段落

### 購買選項
- **顏色選擇器**：按鈕顯示各顏色名稱，點選後主圖切換
- **木材款式選擇器**：按鈕顯示各木材名稱
- **數量選擇器**：- / + 按鈕，最小值 1
- **加入購物車** 按鈕（加入後顯示提示）
- **立即購買** 按鈕（加入購物車後直接導向購物車頁）

### 服務說明列
- 配送：預計 7-14 個工作天到府安裝
- 保固：5 年原廠品質保固
- 退換：30 天鑑賞期，全額退款

### 詳細資訊 Tabs

**規格 Tab**
| 欄位 | 說明 |
|------|------|
| 尺寸 | 長 × 寬 × 高（cm） |
| 重量 | kg |
| 材質 | 材料說明 |
| 原產地 | 製造國家/地區 |
| 設計師 | 設計者姓名 |
| 年份 | 首發年份 |
| 座高 | cm |
| 保固 | 年數 |

**品牌故事 Tab**
- 該商品所屬品牌的詳細介紹段落

**評價 Tab**
- 評價列表（評分 + 評語 + 使用者名稱 + 日期）
- 撰寫評價表單（需登入）：
  - 您的名稱
  - 星級評分（1-5 顆星互動選擇）
  - 評語（多行輸入）
  - 送出按鈕

### 相關商品推薦
- 顯示 4 件隨機商品卡片

## 購物車 `cart.html`

### 購物車列表
- 空車時顯示說明文字 + 「繼續選購」按鈕
- 有商品時顯示：商品圖片、名稱、品牌、顏色、木材、單價、數量（可調整）、小計
- 各商品可單獨刪除
- **清空購物車** 按鈕

### 訂單摘要（右側）
- 小計（NT$）
- 折扣（套用優惠碼後顯示）
- 運費（滿 NT$50,000 免費，否則 NT$3,000）
- 稅金（5%）
- **優惠碼輸入欄 + 套用按鈕**
- **合計**
- **前往結帳** 按鈕

### 運費說明
- 滿 NT$50,000 全台本島免運含到府安裝
- 未達門檻：本島 NT$3,000 / 離島 NT$5,000
- 預計 7-14 個工作天含安裝服務

### 退換貨說明
- 30 天滿意保證，超出法定 7 天猶豫期
- 退貨運費由消費者負擔（NT$3,000–5,000）
- 商品須全新未使用

### 結帳確認 Modal
點擊「前往結帳」後彈出，需填寫：
- 收件人姓名（必填）
- 聯絡電話（必填）
- 收件地址（必填）
- 訂單備註（選填）
- **確認訂單** 按鈕（POST 送出訂單至後端）

## 登入 / 註冊 `login.html`

- 已登入狀態下自動導回首頁
- **登入表單**：Email + 密碼 → POST `/api/auth/login`
- **切換至註冊**：點擊後顯示註冊表單
- **註冊流程**：
  1. 輸入 Email → 點擊「發送驗證碼」
  2. 收到 6 位數驗證碼（有效 10 分鐘）
  3. 填入姓名、密碼、確認密碼、驗證碼
  4. 送出完成註冊

## 管理後台 `admin.html`

需管理員帳號（`ROLE_ADMIN`）才能完整使用。

### 側邊欄
- FORMA Admin Panel 標題
- 導航項目：商品管理 / 訂單查覽 / 品牌管理 / 設計師管理 / 設定
- ← 返回網站

---

### 商品管理

**列表欄位**：縮圖 / 商品名稱+說明摘要 / 品牌 / 類別 / 售價 / 庫存狀態 / 編輯 / 刪除

**新增 / 編輯商品 Modal（大型捲動視窗）**

| 欄位 | 類型 | 說明 |
|------|------|------|
| 商品名稱 | 文字 | 必填 |
| 品牌 | 文字 | 例：Fritz Hansen |
| 設計師 | 文字 | 例：Arne Jacobsen |
| 類別 | 下拉選單 | 椅子 / 沙發 / 桌子 / 收納 / 未分類 |
| 售價 | 數字 | 必填，NT$ |
| 庫存狀態 | Toggle | 有現貨 / 無貨 |
| 首頁展示 | Toggle（3 個） | **本季主打**（精選 tab1）、**設計經典**（精選 tab2）、**Hero 大輪播**（首頁 Banner） |
| 商品說明 | 多行文字 | 材質、尺寸、設計故事 |
| **主圖** | URL 或上傳 | 支援貼上圖片 URL 或點擊上傳本機 JPG/PNG，即時縮圖預覽 |
| **副圖（輪播）** | 多張上傳 | 可逐張新增不同角度照片，顯示於詳情頁縮圖列（少於 2 張自動隱藏縮圖列） |
| **顏色選項** | 動態新增 | 每個顏色變體含：顏色名稱 + HEX 色碼 + 對應圖片（切換顏色時主圖連動） |
| **木材款式** | 動態新增 | 每個木材選項的名稱，顯示於詳情頁的木材選擇按鈕 |

### 設定

- **升級管理員**：輸入目標帳號 Email + 管理員密碼（`FORMA_ADMIN_2025`）即可升級，無需進入資料庫
- **登出後台**：清除 JWT Token，返回一般會員狀態
